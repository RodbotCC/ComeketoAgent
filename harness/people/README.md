# `harness/people/` — contact rolodex

People who aren't (currently) a Close lead but show up across leads:
referrers, vendors, bartenders, photographers, planners, past clients
re-engaging, family members of leads.

## Files

- `{slug}.md` — one per person.

## File shape

```yaml
---
name: Lisa Brennan
role: wedding_planner
email: lisa@...
phone: +1...
referred_leads: [lead_abc, lead_xyz]
last_contact_at: 2026-04-18
---

# Lisa Brennan

(prose: relationship notes, history, voice, etc.)
```

## Status

**Empty.** Populate when a non-lead contact recurs across leads or when
referrals start mattering.
