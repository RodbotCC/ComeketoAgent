Substrate = 200 SEED_FRAMES, each one `{q, comparator, anchor, var_a, var_b, answer}` where answer is a 7-tier choice (exactly_A / more_A / co_equal / blend / more_B / exactly_B / unknown). The `pluck(targetCmp, targetAnchor)` function returns a ranked "corridor" by aggregating three pressure sources: DIRECT plucks (concept literally appeared, 1.0× w/ 1.8× boost on dual match), SIBLING plucks under same comparator (0.35× attenuated by direct evidence), and ANCHOR-AS-PARTICIPANT (anchor's own plucks become a "personal gravity" modifier, 0.45×). Cross-comparator echoes at 0.22×, anchor-match bonus 0.6×. Every answer carries a `trail` of contributing receipts and a `spread` doubling as confidence; abstains below threshold. Self-densifies: every lookup writes DERIVED_FRAMES. Includes ablation (drop a concept, re-ask) and 24-permutation role tests (find stable vs volatile axes).
